Launch Cycle

Look, Listen, and Learn

I've learned a lot about the Python language. I've learned how to use the discord.py framework (which this bot uses), and I've also learned how to code certain functionalities that the commands use, such as choosing a random value from a list or array.

Ask Questions

Some Questions I've asked: How do I grab a random item from an array? How do I organize my commands into separate files? How do I pass in parameters to python functions?

Understand the Process/Problem

To answer my questions, I used a variety of websites, the main ones being StackOverflow and W3Schools. StackOverflow has the answer to nearly every question imaginable, which makes it extremely useful. W3Schools is useful for basic Python fundamentals.

Navigate Ideas

Right from the start, I knew I wanted to make a discord bot. I've been a user of discord for a long time now, and I've seen how beneficial bots can be to discord servers. I also have a passion for coding, and I knew I could learn so much more about Python by taking on this task.

Create a Prototype

I started off by just learning the discord.py framework and how to create a bot account to put online. At this point the bot didn't have any commands.

Highlight and Fix

It took a lot of edits to finally get the bot to where it is now. Much of the problems I encountered dealt with little syntax errors in my code, but there were a few problems that were much larger, such as incorrectly calling commands on an incorrect discord.py class. When encountering these problems, I had to reference back to the discord.py documentation.

Finish

After all my work, I finally present to you: MultiBot.
